= 重設應用程式使用者密碼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
應用程式使用者密碼用於向 vCenter Server 登錄的 SRA 和 VASA Provider 。

.步驟
. 從網路瀏覽器啟動 ONTAP 工具管理員： `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. 使用您在部署期間提供的 VMware vSphere 管理員認證 ONTAP 工具登入。
. 從側邊列選取 * 設定 * 。
. 在 *VASA / SRA 認證 * 畫面中，選取 * 重設密碼 * 。
. 提供新密碼並確認新密碼輸入。
. 選取 * 重設 * 以實作變更。
